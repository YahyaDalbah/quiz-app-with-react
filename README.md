# Quiz-App-With-React
#### Video Demo:  https://youtu.be/5tzNhgtniiA
#### Description:
this project is made with reactjs and css and html
it contains 4 main files

first. `index.js`: it's the place where i render the components to html

second. `App.js`: it contains the API call for the questions and it holded in state (React.useState hook) and it's passing data to question.js 

third. `question.js`: it's the main code source in this project, the questions are displayed in it from `App.js` , it has all of the functionality written in it like checking the answer and display if answer is correct or no

fourth. `style.css`: it styles the project.
